Theme Requirements

In order to proceed with the theme installation, you must be running WordPress 3.3 or higher, PHP 5.3 or higher.

 

Theme Installation via WordPress

  1. Go to Appearance → Themes.
  2. Click Install Themes and hit the upload button
  3. Navigate to find the insomnia.zip file on your computer and click Install Now
  4. Wait while the theme is uploaded and installed.
  5. Activate the newly installed theme. Go to Appearance → Themes and activate it.
  6. After activate, install all plugins from proposed.
 

Theme Installation via FTP

  1. Log into your hosting space via an FTP software.
  2. Unzip the insomnia.zip file and ONLY use the extracted Insomnia theme folder.
  3. Upload the extracted Insomnia theme folder into wp-content → themes folder.
  4. Activate the newly installed theme. Go to Appearance → Themes and activate it.
 

How To Update The Theme

You can update your theme via FTP or via WordPress. See the information below for each method. No matter what method you choose, you first need to download the new theme files from ThemeForest.

 

How To Update Your Theme Via FTP

  1. Go to wp-content → themes location and backup your Insomnia theme folder by saving it to your computer, or you can choose to simply delete it. Your content will not be lost.
  2. Retrieve the insomnia.zip file from your new ThemeForest download (only theme file) and extract the file to get the Insomnia theme folder.
  3. Then simply drag and drop the new Insomnia theme folder into wp-content → themes location. Choose to Replace the current one if you did not delete it.
 

How To Update Your Theme Via WordPress (easy way)

  1. Install Envato Market plugin - This is an easy way to get the latest theme. After you've activate this plugin, Follow these steps.
  2. You will need to generate a personal token, and then insert it in field.
  3. If theme was updated at ThemeForest, you will see notification about new version in this plugin

Standart import demo content

I recommend you to use the demo content files to build a new website which doesn`t have content yet. Our theme includes xml data files which will allow you to upload the demo content to your website. Demo content is the same as in the Live Demo Preview.

  1. Find "Demo" folder in downloaded package from ThemeForest
  2. Find your homepage and choose the your_type_of_content.xml file
  3. Login into WordPress
  4. Go to Tools > Import
  5. Click on "WordPress" If you don't have it already, install the plugin
  6. Click to select file
  7. Choose our your_type_of_content.xml file

NOTICE: Don`t forget to check the "Download and Import File Attachments" in order to import images too.

Portfolio

Insomnia has powerful but very easy for setting portfolio. Only three steps to set up. For detail description see screenshots.

  1. Add VC (Visual Composer) to Portfolio. Go to Visual Composer → Role Manager → Post types: Custom → chose Page and Portfolio
  2. Create New Portfolio work. Go to → Portfolio → Add new
  3. Display Portfolio on site. Navigate to Pages → Add new → Template: Portfolio
  4. Add single work to homepage with VC. Navigate to Pages → Choose your Page → Add Row and Portfolio Item with ID.

Frequently Asked Questions

Q: I want to install "Business" homepage but I got server error.
A:. Any data (page, posts, media, portfolio, etc) comes from my server. My server online all time but your's server (hosting) gives you a limit on uploading files. It's may be limit for request or limit for time. Try to ask to support of your server company.

Q: I waiting about 3-5 minutes for importing data. It's normal?
A:. Yes, all depends on the power of your server.

Q: I imported demo data but got a lot of errors?
A:. Some .xml files contain data from bbPress forum, BuddyPress, WooCommerce, etc. And if you don't install some of this plugin you will get error.

Q: I change color in Customize Panel but some elements stay blue. Why?
A:. You need to change color manually in page with Visual Composer. Choose Elements (for example Button) and tap Edit and change default colors to your.

Q: Sticky menu doesn't work. Why?
A:. Go to Customize > Menu > Change style for activate sticky mode.

Q: I found bug, what I can do?

Q: I have message about activating the plugin (Visual Composer or Revolution Slider)
